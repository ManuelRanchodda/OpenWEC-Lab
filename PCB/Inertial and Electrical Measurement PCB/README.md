# Inertial and Electrical Measurement PCB

This board is responsible for the synchronized acquisition of inertial, mechanical, and electrical variables during laboratory-scale wave energy converter experiments.

## Main Components

- ESP32 Development Board
- 9250 IMU
- Incremental Encoders (N20 motor)
- INA219 Current and Voltage Sensors


## Measured Variables
- Pitch
- PTO Angular Position
- Voltage
- Current
- Electrical Power

## Repository Contents

- Schematics
- PCB Layout
- Gerber Files
- Bill of Materials (BOM)
- Images
- Firmware

This PCB enables synchronized monitoring of the dynamic response of the wave energy converter and the electrical performance of the power take-off (PTO) system.
