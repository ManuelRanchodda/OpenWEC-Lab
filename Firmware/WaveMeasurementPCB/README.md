# Wave Measurement PCB Firmware

This directory contains the firmware developed for the **Wave Measurement PCB** of the OpenWEC-Lab platform.

## Description

This firmware is responsible for acquiring and processing hydrodynamic information during laboratory-scale wave energy converter experiments.

## Hardware

- ESP32 Development Board
- HC-SR04 Ultrasonic Sensor Array

## Functions

- Wave height acquisition
- Sensor synchronization
- Serial data transmission
- Real-time operation

## Output

The firmware streams synchronized wave measurements through the serial interface for further processing and visualization.
